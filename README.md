# DarkSun
Dark fantasy RPG game inspired in old school RPG games without graphic environment.
You are Jacke, a child who suddenly find himself alone in a world that has changed and it is plunged into darkness.

## Commands
### go
  · north/forward <br/>
  · south/back <br/>
  · east/right <br/>
  · west/left <br/>
  
### look

  · east/right <br/>

### equip
